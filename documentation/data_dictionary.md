# Data Dictionary

| Variable | Role | Description |
|---|---|---|
| Diabetes_binary | Target | 0 = no diabetes; 1 = prediabetes or diabetes |
| HighBP | Feature | Ever told that blood pressure is high |
| HighChol | Feature | Ever told that cholesterol is high |
| CholCheck | Feature | Cholesterol checked within the previous five years |
| BMI | Feature | Body mass index |
| Smoker | Feature | Smoked at least 100 cigarettes over lifetime |
| Stroke | Feature | Ever told that the respondent had a stroke |
| HeartDiseaseorAttack | Feature | Coronary heart disease or myocardial infarction history |
| PhysActivity | Feature | Physical activity during the previous 30 days, excluding work |
| Fruits | Feature | Fruit consumed at least once per day |
| Veggies | Feature | Vegetables consumed at least once per day |
| HvyAlcoholConsump | Feature | Heavy alcohol-consumption indicator |
| AnyHealthcare | Feature | Any healthcare coverage |
| NoDocbcCost | Feature | Needed a doctor but could not attend because of cost |
| GenHlth | Feature | Self-rated general health, 1 = excellent through 5 = poor |
| MentHlth | Feature | Days of poor mental health during the previous 30 days |
| PhysHlth | Feature | Days of poor physical health during the previous 30 days |
| DiffWalk | Feature | Serious difficulty walking or climbing stairs |
| Sex | Feature | Binary sex coding in the released dataset |
| Age | Feature | Thirteen-level age-band code |
| Education | Feature | Six-level education code |
| Income | Feature | Eight-level household-income code |

The project preserves the source variable names for reproducibility. See the UCI dataset record for the authoritative source documentation.
